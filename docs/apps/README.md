### Добавление нового приложения

Создайте папку `docs/apps/<slug>/` (латиница, короткий идентификатор):

- **`index.html`** — мини-страница: название, `applicationId`, ссылка на Play, ссылка на `privacy.html`.
- **`privacy.html`** — удобнее хранить **исходник** в этом репозитории приложения как `public/privacy-<slug>.html` и копировать скриптом; либо вручную.

Корень сайта (**один общий файл**):

- `docs/app-ads.txt` — для всех приложений с вашим номером `pub-…` в AdMob достаточно одной строки `google.com, pub-…, DIRECT, …`. Другие партнёрские строки добавляются с новой строки по спецификации IAB.

В **Google Play** для каждого приложения:

- **Веб-сайт разработчика** — общий базовый URL сайта (тот же, что и корень этого Pages).
- **Политика конфиденциальности** — полный URL к `…/apps/<slug>/privacy.html`.
